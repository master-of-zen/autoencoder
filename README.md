# autoencoder

### Usage
`python  autoencoder.py ..args..`

###  CLI parameters
```  --input INPUT, -i INPUT
                        Input File/Folder
  --output OUTPUT, -o OUTPUT
                        Output file name
  --screenshots SCREENSHOTS, -s SCREENSHOTS
                        Number of screenshots to make
  --resolution RESOLUTION [RESOLUTION ...], -r RESOLUTION [RESOLUTION ...]
                        resolutions to encode in```
